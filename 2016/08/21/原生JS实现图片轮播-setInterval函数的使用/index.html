原生JS实现图片轮播--setInterval函数的使用

撰写于 2016-08-21 修改于 2016-08-21 分类 javascript 标签 javascript / 轮播 / 定时器

     之前去面试，面试官要求当场实现轮播，原生js或者jquery都行。只是实现轮播，看到效果就行，先不考虑效果是否好看。jquery里面可能用函数一行代码就可以实现的，我当时却决定用原生js，硬是花了几十分钟，咩写出来，实力尴尬。。。。
     回家后，自己反思当时的思路 咋就不对呢 不应该啊，后来发现一个弱智的问题，解决了，给大家上代码：

lunbo.html页面：

`<!doctype>
<html>
<head>
     <meta  charset="utf-8"/>
      <link rel="stylesheet" href="css/lunbo.css" />
</head>
<body>
     <div   id="container">
            <ul>
                <!-- 设置初始图片的显示 -->
                <li class="active">     
                    <img src="image/content_1.jpg" alt="" />
                </li>
                <li>
                    <img src="image/content_2.jpg" alt="" />
                </li>
                <li>
                    <img src="image/content_3.jpg" alt="" />
                </li>
                <li>
                    <img src="image/content_4.jpg" alt="" />
                </li>
            </ul>
     </div>
</body>
<script type="text/javascript">
    window.onload=function(){
        var index=0;   //初始值
        setInterval(lunbo,1000);    //定时器函数
        function lunbo(){
            //找到所有的图片的li 并存放在数组中
            var arrlist=document.getElementsByTagName('li');   
            arrlist[index].className=""; //
            //判断index值的大小 
            if(index < arrlist.length-1){
                index++;
            }else{
                index=0;
            }
            //index值改变后，设置当前index所属的li的className为active
            arrlist[index].className="active";    
        }

    }

</script>
</html>`

lunbo.css:

`body, ul  ,li,   div  {
   margin:0;
   padding:0;
  }
div{
     border:1px solid #ddd;
     width:733px;
     height:431px;
     margin-left:50px;
}    
ul{
    list-style:none;
}
ul li {
    display:none;
}
/*设置className为active的图片的display属性*/
ul li.active{
    display:block;
 }

`
亲们在复制的时候，注意引用的css的地址，图片地址。不足之处，请大家联系本人进行修改，相互学习，共同进步。
